### 초기 설계 사항

- [ ] 세 가지 랜덤 숫자 생성.
- [ ] 랜덤 숫자 split하여 저장
- 숫자 판독 함수 생성

  - [ ] 볼 : 위치는 다르지만, 존재하는 숫자
  - [ ] 스트라이크 : 존재하고, 위치가 같은 숫자

- [ ] 1을 누르면 게임을 새로 시작.
- [ ] 2를 누르면 게임 종료.

### 예외 사항

1. 사용자가 문자열 입력 시
2. 실수 입력 시
3. 음수 입력 시
4. 3글자보다 작거나 큰 길이의 문자열 입력 시
5. 문자, 숫자 혼용 시 볼, 스트라이크 판정❌ ex) a1c, 12김
